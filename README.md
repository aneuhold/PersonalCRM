# PersonalCRM

## Structure

- Model (Backend)
   - MongoDB with MLab as the basis
- Controller (Backend)
   - Node.js on Heroku free, just 1 server (dyno) is needed. No cost for 1 server running all month.
   - REST API
      - Express
      - Swagger for API front-end
- Front Controller (Front-end)
   - React
- View (Front-end)
   - React
   - Material-UI
